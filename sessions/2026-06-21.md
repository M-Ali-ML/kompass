- **Silent Port Conflicts**: Multiple development instances or orphaned background servers can cause port bind failures (e.g. `Address already in use`). The `dev.sh` startup script now validates port availability upfront.
- **Free Tier Quota Rate Limits**: Real API calls to the Gemini API via the free tier can trigger 429 quota exceptions under load. The frontend correctly falls back to local simulated data when `/api/chat` fails.

## 🧠 Preferences & Architectural Decisions
- **Backend Architecture**: Hexagonal Architecture. The core logic must be isolated from external services. Backend agent logic will use **PydanticAI** with SQLite persistence and Langfuse telemetry.
- **Frontend Framework**: React-based frontend (Next.js/Vite) using **CopilotKit** for chat integration and the **AGUI Protocol** for event-driven UI state streaming (e.g. streaming progress indicators and scenario comparisons).
- **Design System**: Candy design system (rounded shapes, playful vibrant colors, bouncy scales). Avoid basic/default CSS or generic Tailwind styles.
- **Documentation Strategy**: Keep `pre-docs/` as static reference specifications, and maintain `docs/` as a living documentation set updated by agents as code is changed.
- **Git Strategy**: Keep session logs, post-docs, and skill guides checked into Git so they are accessible to any new agent session.
